Android 10

The first beta of Android 10 was released on 13th March 2019 preliminary named Android Q. This beta program was firstly released in Pixel devices. Later on 7th May 2019 Beta 3 was released to other parterned devices. (DSU) Dynamic System Updates, finalized APIs and SDK(API level 29) was released on 5th June 2019 with Beta 4. It was announced on 22nd August 2019 that Android Q would be branded solely as Android 10 with no codename.

Android 10 was officially released on 3rd September 2019. It became the second most popular Android version.

Smart Replay in Notification

Smart Replies now suggest you some smart actions. Messages with url or address can directly be open from the notification itself, no more copying and pasting required.

Gesture Navigation


Full screen gesture mode that eliminates the navigation bar and allows your applications to use the full screen, which enhances the user experience.

Dedicated dark mode for low light and battery saving.

User Privacy

As the privacy of the user is main concern, Android 10 brings some major restriction for the appications using user's location data, device camera, mike etc.

Wifi and QR codes
Android 10 introduces you to wifi QR codes. Connecting to a wifi network by scanning QR code or create your QR code for wifi networks
Google play service System will now fix your security and privacy bugs as your application updates.

Undo the app removal

App removed from the home screen, don't worry here what Android 10 provides. You can undo the app accidentally removed from home.

Live Captions

To make more user friendly interface Android 10 comes with Live Caption. This feature allows the user to turn in caption for any video, audio, podcasts etc. The ML(Machine Learning) Speech mode completely run in the device.

Media


Native MIDI API
Android 10 brings Native MIDI (Musical Instrument Digital Interface) to communicate with the MIDI devices through NDK (Native Development Kit).

Capturing Audio Playback
Capturing audio now made available to any application. Application can catch the audio stream over any other apps using a new audio playback capture API.

Vulkan 1.1
This is a base requirement of Vulkan on device with Android 10 and higher.
Vulkan is nothing but a API for high-performance, 3D Graphics. High-quality, realtime graphics in application.
